# AI Guild -- 3 Month Volunteer Program

## Mission

Build a practical Enterprise Agent Platform while learning LLM basics,
RAG, tool calling, agent patterns, multi-agent systems, and enterprise
AI architecture.

## Engagement Principles

-   Keep weekly scope small
-   Build something tangible
-   Rotate responsibilities
-   Demo every 4 weeks
-   Avoid overload

## Weekly Structure

-   20 min Concept Learning
-   20 min Demo
-   20 min Architecture Discussion

## Role Rotation

-   AI Core Engineer
-   Tool Engineer
-   Guardrail Engineer
-   Documentation Lead

## Month Plan

### Month 1 -- Foundations

LLM Basics, Embeddings, RAG, Evaluation

### Month 2 -- Agents

Tool Calling, ReAct Loop, Planner--Executor, Memory

### Month 3 -- Enterprise Layer

Multi-Agent System, Observability, Guardrails, Service Wrapper

## Success Metrics

-   Working layered platform
-   Guardrails implemented
-   Observability present
-   Final leadership demo
