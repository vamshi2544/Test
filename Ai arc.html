Single ReAct Agent

One LLM. Iterative think → act → observe loop. This is where everyone starts. Build this before touching multi-agent.

User Goal
→
LLM Reasons
→
Tool Call
→
LLM Observes
→
Tool Call
→
Final Answer
Frameworks: LangChain AgentExecutor · LangGraph single node · raw function calling via API
```

Sequential Pipeline

Output of Agent A becomes input of Agent B. Like a data pipeline — each stage is a specialized LLM agent.

Input
→
Researcher Agent
→
Analyst Agent
→
Writer Agent
→
Final Report
Frameworks: CrewAI sequential crew · n8n workflow chains · LangGraph linear graph

Parallel Fan-out / Fan-in

Orchestrator dispatches to multiple agents simultaneously then aggregates results. Reduces total latency significantly.

Orchestrator
→
Agent A — Finance
Agent B — Legal
Agent C — Tech
→
Aggregator
→
Merged Output
Frameworks: LangGraph parallel nodes · AutoGen GroupChat · CrewAI concurrent tasks

Supervisor–Worker (Hierarchical)

Supervisor plans and delegates. Workers execute. Supervisor evaluates and re-routes if needed. Most common production pattern.

Goal
→
Supervisor LLM
⇄
Coder Worker
QA Worker
Docs Worker
→
Deliverable
Frameworks: CrewAI hierarchical process · LangGraph supervisor pattern · AutoGen nested chats

Reflection / Agent Debate (Critic Pattern)

Multiple agents critique each other's outputs in a loop. Reduces hallucinations and improves quality for high-stakes tasks.

Generator Agent
→
Critic Agent
→
Refiner Agent
→
Critic Agent
→
Final Output
Frameworks: AutoGen AssistantAgent + CriticAgent · LangGraph reflection loops · self-reflection prompting

// Learning Roadmap — End to end from zero to production agents
```
Level 0

Foundations

What you need to understand before writing any agent code
Topics
  • How LLMs work — tokens, context window, temperature
  • What is a prompt? System vs user vs assistant roles
  • Calling AI APIs — OpenAI, Anthropic (Claude)
  • JSON, environment variables, .env files
  • Git basics — clone, commit, push
Level 1

Python for AI

Just enough Python to build AI pipelines — not a full Python course
Topics
  • Variables, loops, functions, dicts, lists
  • Installing packages with pip, virtual environments
  • Making API calls with requests / httpx
  • Reading and writing files, parsing JSON
  • Async / await basics — needed for agent frameworks
  • Using .env with python-dotenv
Prereq → Level 0
Level 2

Prompt Engineering

The skill that connects your intent to model behavior — critical before agents
Topics
  • Zero-shot, few-shot, chain-of-thought prompting
  • System prompts — persona, constraints, output format
  • Structured output — JSON mode, XML tags
  • Prompt templates with variables
  • Iterative prompt debugging and testing
Prereq → Level 1
Level 3

RAG — Retrieval-Augmented Generation

Give your LLM access to your own knowledge. Foundation for agent memory.
Topics
  • What are embeddings and vector databases
  • Document ingestion — load, chunk, embed, store
  • Semantic search — query → retrieve → inject into prompt
  • ChromaDB / Pinecone hands-on
  • Build a Q&A bot over your own documents
  • Hybrid search — keyword + semantic combined
Prereq → Level 2
Level 4

Single Agent — ReAct

Build your first real agent. One LLM, tools it can call, iterative reasoning loop.
Topics
  • Agent vs chatbot vs chain — what is the difference
  • Tool / function calling — define, register, invoke
  • ReAct loop — Reason, Act, Observe, repeat
  • Build agent with LangChain AgentExecutor
  • Add memory — short-term conversation history
  • Intro to LangGraph — nodes, edges, state
  • Debugging and tracing with LangSmith
Prereq → Level 3
Level 5

Multi-Agent Systems

Multiple agents collaborating, each with a role. This is the real power of agentic AI.
Topics
  • Agent roles — Planner, Executor, Critic, Router
  • Sequential pipeline pattern
  • Supervisor–Worker pattern with LangGraph
  • Parallel fan-out / fan-in
  • CrewAI — define crew, roles, tasks
  • AutoGen — conversational multi-agent
  • Agent communication — shared state, message passing
  • Routing logic — how agents hand off to each other
Prereq → Level 4
Level 6

Production-Ready Agents

Take agents from prototype to deployable, observable, and safe systems.
Topics
  • Guardrails — input validation, output schema enforcement
  • PII detection and masking
  • Hallucination detection — LLM-as-judge pattern
  • Full observability with LangSmith / Langfuse
  • Token cost tracking and budget limits
  • Error handling, retries, fallback strategies
  • Containerize and deploy — Docker + cloud
  • Evaluation — how to test and benchmark your agent
Prereq → Level 5
Level 7

Build Something Real

Apply everything end-to-end by building a real agentic product from scratch.
Topics
  • Pick a problem domain — internal tool, SaaS, automation
  • Design the agent architecture from scratch
  • Choose the right pattern for your use case
  • Integrate with real data sources and APIs
  • Add human-in-the-loop checkpoints
  • Deploy, monitor, iterate on real usage
Prereq → Level 6
