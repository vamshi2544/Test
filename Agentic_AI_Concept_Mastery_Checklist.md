# Agentic AI Concept Mastery Checklist

## Layer 1 -- LLM Foundations

-   [ ] I can explain what a transformer is (high level).
-   [ ] I understand tokens and context window limits.
-   [ ] I can explain why LLMs hallucinate.
-   [ ] I understand temperature and sampling.
-   [ ] I can design structured prompts with clear system/user
    separation.

## Layer 2 -- RAG & Retrieval

-   [ ] I understand what embeddings are.
-   [ ] I can explain cosine similarity.
-   [ ] I understand chunking strategies and tradeoffs.
-   [ ] I can explain full RAG flow (retrieve → inject → generate).
-   [ ] I understand RAG failure modes.

## Layer 3 -- Agents & Tool Use

-   [ ] I can define what an agent is (LLM + tools + loop).
-   [ ] I understand function/tool calling conceptually.
-   [ ] I can explain the ReAct pattern.
-   [ ] I understand planner--executor architecture.
-   [ ] I understand different types of agent memory.

## Layer 4 -- Multi-Agent Systems

-   [ ] I understand role-based agent specialization.
-   [ ] I understand coordination and message passing.
-   [ ] I can explain conflict resolution strategies.
-   [ ] I understand autonomy spectrum (human-in-loop vs fully
    autonomous).

## Layer 5 -- Enterprise AI Architecture

-   [ ] I understand prompt injection risks.
-   [ ] I understand tool misuse risks.
-   [ ] I can explain guardrails and policy enforcement.
-   [ ] I understand AI observability concepts.
-   [ ] I understand AI evaluation frameworks.
-   [ ] I understand governance and compliance considerations.
